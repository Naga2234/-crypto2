== Description ==

WP Security Mini - минималистичный плагин безопасности с максимальной эффективностью:

✅ **DDoS защита**
* Автоматическое определение массовых запросов
* Challenge-страница при подозрительной активности
* Автоматическая блокировка при превышении лимитов

✅ **Защита от брутфорса**
* Блокировка после 5 неудачных попыток входа
* Автоматическая разблокировка через 30 минут

✅ **Блокировка вредоносных запросов**
* SQL injection
* XSS атаки
* Directory traversal
* Code injection

✅ **Топ-10 атакующих IP**
* Анализ самых активных злоумышленников
* Статистика за последние 7 дней
* Возможность ручной блокировки

✅ **История входов**
* Все попытки входа (успешные и неудачные)
* Определение устройства и браузера
* IP адреса и временные метки

✅ **Оптимизация**
* Минимальная нагрузка на сервер
* Автоматическая очистка старых данных
* Хранение только последних 500 записей истории
* Без внешних зависимостей

== Installation ==

1. Загрузите папку плагина в `/wp-content/plugins/`
2. Активируйте плагин через меню "Плагины"
3. Перейдите в "Security" для просмотра статистики
4. Настройте параметры в разделе "Настройки"

== Frequently Asked Questions ==

= Влияет ли плагин на производительность сайта? =

Нет, плагин оптимизирован для минимального влияния. Использует индексы БД и автоматическую очистку данных.

= Как изменить порог срабатывания DDoS защиты? =

Перейдите в Security → Настройки и измените значение "Порог срабатывания".

= Сколько хранятся логи? =

По умолчанию 14 дней. Можно изменить в настройках от 7 до 90 дней.

= Можно ли восстановить заблокированный IP? =

Да, на странице Dashboard есть кнопка "Разблокировать" для каждого IP.
